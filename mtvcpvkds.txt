📌 PLATAFORMAS UTILIZADAS NO PROJETO
====================================================================

📁 **GITHUB — REPOSITÓRIO PRINCIPAL DO PROJETO**
É onde todo o código e conteúdo do sistema fica armazenado e organizado.

**Arquivos principais na pasta raiz:**
- `mtvcpvkds.txt` → Regras, especificações e estrutura completa do sistema
- `index.html` → Tela inicial do aplicativo (interface principal)
- `vercel.json` → Configuração de hospedagem e rotas de navegação
- `manifest.json` → Configuração para funcionar como aplicativo instalável (PWA)
- `/src` → Páginas, telas e funcionalidades internas do sistema
- `/services` → Integrações com IA, banco de dados e processamento de pagamentos

**Funções desta plataforma:**
✔ Criação e edição de todo o conteúdo e código
✔ Controle de versões e alterações
✔ Organização completa da estrutura do projeto
✔ Integração automática com a plataforma de hospedagem Vercel

====================================================================

🌐 **VERCEL — HOSPEDAGEM E EXECUÇÃO DO SISTEMA**
Plataforma onde o aplicativo fica disponível online e acessível ao público.

**Funções principais:**
- Hospedagem do site e aplicativo no formato PWA
- Publicação automática da versão mais recente vinda do GitHub
- Configuração de endereço público e certificado de segurança HTTPS automático
- Armazenamento seguro de variáveis de ambiente e chaves secretas
- Atualização por meio da função **Reimplantar** sempre que houver alterações

**Endereço oficial do aplicativo:**
`https://motivacoespravidakidsmd.vercel.app`

**Funções desta plataforma:**
✔ Colocar o sistema no ar para acesso público
✔ Garantir funcionamento em navegadores de celular, tablet e computador
✔ Habilitar instalação como aplicativo nativo nos aparelhos

====================================================================

🧠 **SUPABASE — BANCO DE DADOS + AUTENTICAÇÃO**
Responsável por todo o gerenciamento de dados, cadastro e acesso seguro do sistema.

**Tabelas obrigatórias configuradas:**
- `users` → Dados dos responsáveis e status do plano contratado
- `stories` → Histórias criadas por cada usuário
- `consent` → Registro da concordância com termos e regras de privacidade
- `subscriptions` → Controle de validade, status e renovação de assinaturas

**Funções desta plataforma:**
✔ Sistema de login e autenticação segura
✔ Armazenamento permanente de todo o conteúdo gerado
✔ Controle automático de permissões e tempo de acesso
✔ Proteção e criptografia dos dados conforme a LGPD

**Regra de integração:**
- Endereço e chaves de acesso são cadastrados **apenas como variáveis na Vercel**
- Nunca são mostrados ou armazenados nos arquivos visíveis no GitHub

====================================================================

🤖 **OPENAI — INTELIGÊNCIA ARTIFICIAL**
Responsável pela criação inteligente de todo o conteúdo personalizado.

**Funções principais:**
- Geração de histórias adaptadas por idade e perfil da criança
- Criação de textos educativos, positivos e adequados ao público infantil
- Geração de ilustrações com estilo visual definido e sem conteúdo inadequado

**Configuração de segurança:**
- Chave de API gerada na conta da OpenAI
- Cadastrada exclusivamente como variável de ambiente na Vercel
- Nunca incluída diretamente no código ou arquivos do repositório

**Funções desta plataforma:**
✔ Criação de experiências únicas para cada criança
✔ Geração automática de conteúdo consistente com as escolhas feitas
✔ Ilustrações com padrão seguro e aprovado para o projeto

====================================================================

💳 **HOTMART — VENDAS E MONETIZAÇÃO**
Plataforma responsável pelo processamento de pagamentos e liberação de acesso.

**Configurações obrigatórias:**
- Produtos cadastrados conforme os planos definidos: 15 dias, mensal e anual
- Endereço de **Webhook** configurado para receber confirmações de compra
- Regras de envio de dados para integração com o Supabase

**Fluxo completo de funcionamento:**
1. Responsável realiza a compra do plano escolhido
2. Hotmart confirma e aprova o pagamento
3. Sinal automático é enviado pelo Webhook
4. Supabase registra e libera o acesso correspondente
5. Usuário já pode entrar e usar o sistema

**Funções desta plataforma:**
✔ Processamento seguro de pagamentos por Pix, cartão e outros meios
✔ Liberação automática logo após aprovação
✔ Controle da validade e renovação de planos
